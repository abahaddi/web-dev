---

- Responsive design
- Hover animations

---

## ⚙️ CI/CD Pipeline

This project includes a CI/CD pipeline using **GitHub Actions**.

Each time a change is pushed to the repository, the pipeline automatically:

1. Builds a Docker image of the project
2. Pushes the image to the workflow
3. Deploys it to an **AWS EC2 instance**
4. Uses a self-hosted GitHub Actions runner on EC2

This means every update to the code automatically triggers a new deployment.

---

## 🔄 How it works

- Push code to GitHub
- GitHub Actions pipeline is triggered
- Docker image is built
- New version is deployed automatically to EC2
